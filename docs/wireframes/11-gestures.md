### 7.2 Conflict Resolution

| Conflict | Resolution |
|----------|------------|
| Edge swipe vs horizontal scroll | Edge zone (20pt) for back gesture |

| Pinch vs scroll | Multi-touch = pinch priority |
| Chapter swipe vs list scroll | Horizontal vs vertical velocity |

### 10.1 iOS vs Android

| Gesture | iOS | Android |
|---------|-----|---------|
| Back navigation | Edge swipe from left | System back button/gesture |
| Bottom sheet | Native sheet behavior | Bottom sheet pattern |
| Haptics | Taptic Engine | Vibration patterns |
| Long press duration | 500ms | 500ms |
| Edge zones | 20pt | Respects system gesture areas |

### 10.2 Platform-Specific Considerations

```
iOS:
- Respect safe areas for gestures
- Use SF Symbols haptic patterns
- Home indicator zone (bottom 34pt) reserved

Android:
- Navigation gesture bar at bottom
- Predictive back animation support
- Back gesture from both edges (Android 13+)
```
