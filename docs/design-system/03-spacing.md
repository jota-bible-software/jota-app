# Design System: Spacing & Layout

**Document Version**: 1.0
**Last Updated**: December 2024

---

## Overview

This document defines the spacing system, layout grids, and dimensional specifications for the Jota mobile app.

---

## 1. Spacing Scale

### 1.1 Base Unit

```
BASE UNIT: 4pt

All spacing values are multiples of 4pt
This creates visual rhythm and consistency
```

### 1.2 Spacing Scale

| Token | Value | Use Case |
|-------|-------|----------|
| space-0 | 0pt | No spacing |
| space-1 | 4pt | Tight, inline elements |
| space-2 | 8pt | Related elements |
| space-3 | 12pt | Form fields, list items |
| space-4 | 16pt | Standard padding |
| space-5 | 20pt | Section spacing |
| space-6 | 24pt | Card padding |
| space-7 | 28pt | Large gaps |
| space-8 | 32pt | Section separators |
| space-10 | 40pt | Large separators |
| space-12 | 48pt | Screen margins (large) |
| space-16 | 64pt | Hero spacing |
